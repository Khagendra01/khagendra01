# Algorithm Tester Website

A Streamlit application to test machine learning algorithms for keystroke dynamics and text analysis.

## Structure

- `numb_app.py`: Analyzes numeric input patterns.
- `text_app.py`: Analyzes text input patterns.
- `*.h5`: Pre-trained TensorFlow/Keras models.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   streamlit run numb_app.py
   # or
   streamlit run text_app.py
   ```

## Note

This application requires specific JSON input files generated by the data collection tool.
